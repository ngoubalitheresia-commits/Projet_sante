
# Projet Santé Publique - Akieni Academy

Ce projet simule un système de gestion de données hospitalières et pharmaceutiques.  
Il centralise les constantes OMS, les informations de 5 hôpitaux et 5 médicaments essentiels.  
Des indicateurs clés (KPI) sont calculés pour analyser la performance du système de santé.  
Projet évolutif sur plusieurs semaines dans le cadre du programme Fil Rouge numéro 1 Sante Publique.
